#  AI WhatsApp Assistant

An AI-powered WhatsApp chatbot built using Python, Flask, Twilio, ngrok, and Groq LLM.

This project receives WhatsApp messages in real time, processes them using AI, and sends intelligent replies back to the user.

---

#  Features

- Real-time WhatsApp messaging
- AI-generated responses
- Flask backend integration
- Twilio WhatsApp Sandbox support
-  Groq LLM integration
-  Easy local deployment with ngrok

---

#  Tech Stack

| Technology | Purpose |
|---|---|
| Python | Backend programming |
| Flask | Web server |
| Twilio API | WhatsApp integration |
| ngrok | Public tunnel for local server |
| Groq LLM | AI response generation |
